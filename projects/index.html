A few things I've built and written that I'm proud of — mostly focused on making information easier to use, and experiences more intuitive.

Messaging App for Planning + Groups

A chat-style app built around real needs: making it easier for groups to plan events, coordinate, and keep conversations organized without chaos.

Online Board Game

A web-based board game designed to feel smooth and fair, even when multiple people are playing at the same time. I cared a lot about clarity and making it easy to use.

Satellite Communication Write-Up

A proposal that lays out how satellites could reliably talk to each other. I focused on tricky situations (when things go wrong) and the tradeoffs between speed, reliability, and simplicity.

NLP-Powered Research Finder

A project at J-PAL to help people find the right researchers and topics faster using plain-language search — so evidence is easier to discover and easier to use.
